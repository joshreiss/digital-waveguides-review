STK Clarinet - Web Audio Implementation

Web Audio API implementation of the Clarinet physical model from the Synthesis ToolKit (STK). It uses a digital waveguide model with a nonlinear reed table and feedback loop to simulate a clarinet.

Click "Start Clarinet" button, then play using mouse or computer keyboard. Keyboard mapping: A = C4, S = D4, D = E4, F = F4, G = G4, H = A4, J = B4, K = C5
