Faust Clarinet - Web Audio Implementation

This is a Web Audio API implementation based on the Faust Clarinet physical model by Romain Michon. It uses a digital waveguide model with a non-linear reed function, feedback loop, and stereo spatialization. This implementation follows the parameter organization and UI design of the original Faust DSP code.

Click the "Start Clarinet" button below, then play using your mouse or computer keyboard.
